Varia

  • HJB - Showcase for a static and easy implementation of a possible digital edition of Hjertebogen. The implementation is based on R-markdown and uses the R-package flexdashboard for the sake of layout and style.

  • Sovfilms Network - A Plotly-Plot of a Network for “Who plays with whom in Soviet films?” (“Wer spielt mit wem in sowjetischen Filmen?”). A university project. The files are available on my Github profile. The network layout is Fruchterman-Reingold (with basic paramaters, as in nx.spring_layout(G)).
